# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère à [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Ajouté
- Structure de projet React professionnelle avec TypeScript
- Système de design avec composants UI réutilisables (Button, Input, Card)
- Gestion d'état avec Zustand et persistance localStorage
- Pages principales : Accueil, Historique, Statistiques
- Formulaire d'ajout de biberons avec quantité, type et notes
- Calcul automatique des statistiques (jour, semaine, mois)
- Navigation entre les pages
- Formatage de dates en français
- Configuration ESLint et Prettier
- Variables CSS pour thème cohérent
- Support automatique du mode sombre/clair

### Configuration
- Vite pour le build
- TypeScript avec configuration stricte
- ESLint pour la qualité du code
- Prettier pour le formatage
- EditorConfig pour la cohérence
- Configuration VS Code recommandée
